# Tecnomultimedia

El escenario en el que se encuentra el personaje seria una casa normal el acaba de morir y lo que quiere es tener su descanso eterno de su alma.
Pensaba hacer una especie de final secreto en el que se puedan encontrar los 2 elementos claves para la trama y apareza un personaje secreto con un dialogo distinto, que este vivo quiza.
